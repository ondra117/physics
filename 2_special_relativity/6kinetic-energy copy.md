$$
dp = dp\\
dp = \frac{dp}{dv}dv\\
\frac{dp}{dv} = \frac{d}{dv}mv\gamma = m\gamma + mv\frac{d}{dv}\frac{1}{\sqrt{1-\frac{v^2}{c^2}}} = m\gamma + mv\frac{v}{c^2\left(1 - \frac{v^2}{c^2}\right)^{\frac{3}{2}}} = m\gamma + m\frac{v^2}{c^2}\gamma^3 = m\gamma\left(1 + \frac{v^2}{c^2}\gamma^2\right) = m\gamma\left(1 + \frac{v^2}{c^2}\frac{1}{1-\frac{v^2}{c^2}}\right) = m\gamma\left(1 + \frac{\frac{v^2}{c^2}}{1-\frac{v^2}{c^2}}\right) = m\gamma\frac{1-\frac{v^2}{c^2} +\frac{v^2}{c^2}}{1-\frac{v^2}{c^2}} = m\gamma\frac{1}{1-\frac{v^2}{c^2}} = m\gamma^3\\
\frac{dp}{dv} = m\gamma^3
$$
$$
K = \int v\ dp = \int v\frac{dp}{dv}\ dv = \int vm\gamma^3\ dv = \int v\frac{dp}{dv}\ dv = \int mv\frac{1}{\left(1-\frac{v^2}{c^2}\right)^{\frac{3}{2}}}\ dv = m\int \frac{v}{\left(1-\frac{v^2}{c^2}\right)^{\frac{3}{2}}}\ dv = mc^2\int \frac{v}{c^2\left(1-\frac{v^2}{c^2}\right)^{\frac{3}{2}}}\ dv = \left[mc^2\gamma\right]_0^v = mc^2\gamma(v) - mc^2\gamma(0) = mc^2\gamma - mc^2 = (\gamma - 1)mc^2
$$
$$
