# Vpřed
$$
t_0 = 0 + \frac{X - 0}{c} = \frac{X}{c}\\
t_1 = \gamma\Delta t^\prime + \frac{X - \gamma v\Delta t^\prime}{c}\\
\Delta t = t_1 - t_0 = \gamma\Delta t^\prime + \frac{X - \gamma v\Delta t^\prime}{c} - \frac{X}{c} = \gamma\Delta t^\prime - \frac{\gamma v\Delta t^\prime}{c} = \gamma\Delta t^\prime\left(1 - \frac{v}{c}\right)\\
\Delta t = \gamma\Delta t^\prime\left(1 - \frac{v}{c}\right)\\
f = \frac{1}{\Delta t} = \frac{1}{\gamma\Delta t^\prime\left(1 - \frac{v}{c}\right)} = \frac{1}{\gamma\frac{1}{f^\prime}\left(1 - \frac{v}{c}\right)} = \frac{f^\prime}{\gamma\left(1 - \frac{v}{c}\right)}\\
E^{\prime} = E\frac{f^{\prime}}{\frac{f^\prime}{\gamma\left(1 - \frac{v}{c}\right)}} = E\gamma\left(1 - \frac{v}{c}\right)
$$
# Vzad

E^{\prime} = E\frac{f^{\prime}}{\frac{f^\prime}{\gamma\left(1 + \frac{v}{c}\right)}} = E\gamma\left(1 + \frac{v}{c}\right)
